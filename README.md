# sche_do_project
Project Application : Task Management

#  How to Run
1. git clone repo
2. push PHP at `htdocs/sche_do_project/`
3. save `pubspec.yaml` to fix error
4. open **XAMPP** and Start 'Apache' and 'MySQL' while run
   
   
**Check API:** [http://localhost/sche_do_project/backend_api/get_tasks.php](http://localhost/sche_do_project/backend_api/get_tasks.php)
